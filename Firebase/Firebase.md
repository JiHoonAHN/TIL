# Firebase

Firebase SDK 

> SDK 란 **Software Devleopment Kit** 이라는 말의 줄임말으로 간단하게 말하면 개발 도구리고 보면 된다. 그래서 Firebase iOS SDK는 Fibase기능을 iOS에 접목하기 위한 개발도구가 있다고 생각하면 된다.

<br>

SDK를 cocoapods으로 가져올수 있다.
